# 🎬 Flask Movie App

A full-stack movie rating web application built using Flask.  
Users can search, add, rate, review, and manage their favorite movies.

---

## 🚀 Features

- 🔍 Search movies using TMDB API
- ➕ Add new movies
- ⭐ Rate movies
- ✍️ Write reviews
- 🗑️ Delete movies
- 🏆 Automatic ranking system (based on ratings)

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask
- **Database:** SQLite, SQLAlchemy
- **Frontend:** HTML, CSS, Bootstrap
- **API:** TMDB (The Movie Database API)

---

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/flask-movie-app.git
cd flask-movie-app

Create virtual environment:
python -m venv venv
venv\Scripts\activate   # Windows

Install dependencies:
pip install -r requirements.txt

Add your TMDB API key in main.py:
TMDB_API_KEY = "YOUR_API_KEY"

Get API key from: https://www.themoviedb.org/settings/api

Run the application:
python main.py
